void altera_tabuleiro ( ESTADO *  estado)

Função que preenche o tabuleiro de acordo com as jogadas efetuadas até a n-ésima jogada, onde n é o número de jogadas atual. Útil utilizá-la sempre que é efetuada uma jogada de modo a se preocupar apenas em alterar o número de jogadas mediante a necessidade, podendo ser uma função reutilizada tanto para o comando se jogar normal como também após ter sido efetuado um comando "pos" .

estadoAppontador para o estado do jogo
